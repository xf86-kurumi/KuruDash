# DashLab
DashLab is a beautiful, fully local homelab service dashboard. No backend. No database. No tracking. Just one HTML file. 

# Features:

- **8 built-in themes** — Frutiger Aero, Dark, Catppuccin Mocha, Nord, Solarized, Sakura, Terminal, Flat Dark/Light
- **Custom theme builder** — pick your own colors with a live color picker
- **Auto icon fetching** — icons are grabbed automatically from your services' favicons
- **SSH support** — copy SSH commands or open via `ssh://` protocol handler
- **Status checks** — live up/down indicators for all your services
- **Folder organization** — group services into collapsable folders
- **Sync / Backup** — export and import all your data as a single JSON file
- **100% local** — all data stored in browser localStorage, nothing sent anywhere
- **Custom UI icons** — drop your own PNGs in `icons/` to replace the default UI icons
- **Responsive** — works on desktop and mobile
